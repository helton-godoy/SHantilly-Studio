# Política de Segurança

## Versões Suportadas

O suporte de segurança segue o ciclo de lançamento principal:

| Versão | Suportada          |
| ------ | ------------------ |
| Main   | :white_check_mark: |
| Stable | :white_check_mark: |

## Reportando uma Vulnerabilidade

Se você descobrir uma falha de segurança no SHantilly-Studio:

**NÃO crie uma Issue pública.**

Por favor, reporte a vulnerabilidade de forma privada:

* Email: <contact@shantilly.org>
* Ou via mensagem direta para os mantenedores.

### Processo

1. Analisaremos o relatório.
2. Desenvolveremos uma correção em ambiente privado.
3. Lançaremos uma nova versão com a correção e creditaremos você (se desejar).

Obrigado pela colaboração responsável!
