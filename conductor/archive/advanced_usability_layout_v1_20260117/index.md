# Track: Advanced Usability & Layouts

## Context

This track aims to transform Showbox Studio from a linear editor into a full-featured visual designer with Undo/Redo support and advanced layout management.

## Goals

1.  **Undo/Redo:** Robust history navigation.
2.  **Reordering:** Drag & Drop within the hierarchy.
3.  **Grouping:** Easy creation of complex nested layouts.
