# Track: Studio Persistence & Code Generation

## Context
This track focuses on enabling the Showbox Studio to save projects and export executable shell scripts.

## Goals
1.  **Code Export:** Generate `.sbx` scripts compatible with Showbox CLI.
2.  **Project Save/Load:** Persist editor state using JSON.
3.  **Live Preview:** Run the generated script instantly.
